# AI-safe protocol

Раздел нужен, чтобы получать помощь от AI без выноса внутренних данных, секретов и корпоративной информации.

## Что можно передавать

- Обезличенное описание задачи.
- Синтетические примеры входных данных.
- Структуру JSON/YAML/CSV без реальных значений.
- 1–10 строк логов после очистки.
- Технические требования без внутренних доменов, IP, токенов, ФИО.

## Что нельзя передавать

- Пароли, токены, cookies, private keys.
- Реальные домены, внутренние URL, VPN-адреса.
- Реальные IP внутренней сети.
- ФИО пользователей, почты, телефоны.
- Полные конфиги production-сервисов.
- Дампы баз, логи с персональными данными.
- Код целиком, если он считается внутренней интеллектуальной собственностью.

## Универсальная замена

```text
real-company.local       -> example.local
service.real-company.ru  -> service.example.local
10.100.143.86            -> 10.0.0.10
DC02                     -> dc01
srv-backupdata           -> fileserver01
ivanov.i                 -> user1
TechSupport              -> support
Authorization: Bearer X  -> Authorization: Bearer <TOKEN>
Cookie: abc              -> Cookie: <COOKIE>
```
