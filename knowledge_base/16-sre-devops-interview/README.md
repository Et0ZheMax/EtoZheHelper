# SRE / DevOps / Sysadmin interview

## Как позиционироваться

Ты не "чистый разработчик". Сильная позиция:

```text
Infrastructure/System Administrator с уклоном в automation и ростом в DevOps/SRE.
Сильная база: Linux/Windows/AD/DNS/SSH/CUPS/GLPI/troubleshooting.
Развиваю Docker, Ansible, CI/CD, monitoring, Kubernetes, Terraform.
```

## Что повторить

- Linux process/systemd/logs/filesystems;
- DNS/TCP/IP;
- Docker/Compose;
- CI/CD stages;
- Ansible;
- monitoring/logging;
- Bash/Python basics;
- incident troubleshooting.
